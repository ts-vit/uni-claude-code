# @uni-fw/ssh-ui

SSH tunnel settings UI component for [UNI Framework](https://github.com/ts-vit/ai-chat).

## Features

- `SshTunnelSettings` — complete SSH tunnel configuration form
- Host, port, username, authentication (password/key)
- Connection status display
- Connect/disconnect controls
- Uses `@uni-fw/ui` settings adapter and `@tauri-apps/api` for backend communication

## Installation
```bash
npm install @uni-fw/ssh-ui
```

### Peer Dependencies

- `react` >= 19
- `@mantine/core` >= 8
- `@uni-fw/ui`
- `@tauri-apps/api` >= 2
