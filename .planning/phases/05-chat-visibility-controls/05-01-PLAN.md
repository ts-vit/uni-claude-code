- **D-05-14** — `chat.cleared` остаётся dead-key.
- **D-05** Claude's Discretion → новый параллельный тип `SessionMetadata`, а не расширение `SessionResult` (раздельные семантики «live runtime state» vs «итоги завершённой сессии»).

Output: `src/types/claude.ts` с новым экспортом `SessionMetadata`; `src/i18n/locales/en.json` и `src/i18n/locales/ru.json` с расширенным `chat.*` блоком. `npm run typecheck` и `npm run test` обязаны проходить (никаких реальных consumer'ов нового типа ещё нет — план purely additive).
